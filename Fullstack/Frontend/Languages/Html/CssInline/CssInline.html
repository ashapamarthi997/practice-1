INLINE CSS

1.we can apply CSS in a single element by inline CSS technique.

2.he inline CSS is also a method to insert style sheets in HTML document.

3.This method mitigates some advantages of style sheets so it is advised to use this method sparingly.

4.If you want to use inline CSS, you should use the style attribute to the relevant tag.

Syntax:

htmltag style="cssproperty1:value; cssproperty2:value /htmltag

Disadvantages of Inline CSS

*.You cannot use quotations within inline CSS.

*.If you use quotations the browser will interpret this as an end of your style value.

*.These styles cannot be reused anywhere else.
