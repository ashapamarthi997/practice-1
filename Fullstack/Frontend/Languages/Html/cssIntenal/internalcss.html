Internal CSS

1.The internal style sheet is used to add a unique style for a single document.

2.it is defined in HEAD section of the HTML page inside the STYLE tag.

Example:
!DOCTYPE html
html
head
style
body {
background-color: linen;
}
h1{
color: red;
margin-left: 80px;
}
/style>
/head>

The internal style sheet is applied on this heading.

p>This paragraph will not be affected./p>
