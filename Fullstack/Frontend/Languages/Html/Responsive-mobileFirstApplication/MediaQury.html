The Media query in CSS is used to create a responsive web design.

means that the view of a web page differs from system to system based on screen or media types. The breakpoint specifies for what device-width size, the content is just starting to break or deform.

Media queries can be used to check many things:

*.width and height of the viewport
*.width and height of the device
*.Orientation
*.Resolution
